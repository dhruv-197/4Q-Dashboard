## 🚀 Getting Started

Follow the steps below to run this project locally.

### 1. Clone the Repository
```bash
git clone https://github.com/your-username/your-repo-name.git
cd your-repo-name
```

### 2. Install Dependencies
```bash
npm install
```

### 3. Start the Development Server
```bash
npm run dev
```

### 4. Open in Browser
After running the command above, open the URL shown in your terminal.  
It will usually be: http://localhost:5173
